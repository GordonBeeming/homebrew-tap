# Homebrew Tap

Custom Homebrew formulae by [Gordon Beeming](https://github.com/GordonBeeming).

## Installation

```bash
brew tap gordonbeeming/tap
```

## Available Formulae

### copilot_here

Run GitHub Copilot CLI in a sandboxed Docker container.

```bash
brew install gordonbeeming/tap/copilot_here
```

For more details, see the [copilot_here repository](https://github.com/GordonBeeming/copilot_here).
